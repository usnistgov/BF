    BF Publications
    Irena Bojanova, Inventor/Creator, PI & Lead, NIST Bugs Framework (BF), 2014 – ~~~ #

    Please, reference BF publications recent than year 2020, as some old BF classes are now covered by new more comprehensive classes and some – are withdrawn. They also start developing my ideas on the BF's formalism – originally shared in my 2014/2015 presentations. Please use as source about the BF class taxonomies this website and the publications listed here – any other publications may provide misleading information. If in doubt, please seek guidance from the BF PI .
